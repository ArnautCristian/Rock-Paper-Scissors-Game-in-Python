# Rock Paper Scissors Game 🎮

A Python command-line Rock Paper Scissors game where the player competes against the computer over multiple rounds.

The game allows the user to choose the number of rounds, validates user input, keeps track of scores, and announces the final winner.

## Features

- Choose the number of rounds:
  - 3 rounds
  - 5 rounds
  - 7 rounds
- User input validation
- Error handling for invalid numerical inputs
- Computer opponent with randomly generated choices
- Score tracking for both player and computer
- Displays each round's choices
- Replay option after finishing a game

## How It Works

1. The player chooses how many rounds they want to play.
2. Each round:
   - The player chooses rock, paper, or scissors.
   - The computer randomly selects its choice.
   - The program determines the winner of the round.
   - The score is updated.
3. After all rounds are completed, the final score is displayed.
4. The player can choose whether to play again.

## Technologies Used

- Python 3
- `random` module
